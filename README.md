# My UI Components Website

Это личный проект, в котором собраны мои собственные **UI-компоненты** для веб-разработки. Сайт демонстрирует разные элементы интерфейса, которые я создаю на React и стилизую с помощью Tailwind CSS и кастомных стилей.

---

## Цель проекта

- Создать библиотеку переиспользуемых UI-компонентов.  
- Показать навыки в верстке, React и стилизации.  
- Сделать удобный и красивый интерфейс для демонстрации компонентов.  

---

## Особенности

- **Кастомные компоненты:** заголовки, описания, кнопки, формы, карточки и т.д.  
- **Современный дизайн:** адаптивные элементы с hover-эффектами, анимациями и плавными переходами.  
- **Tailwind CSS + SCSS:** комбинирую утилитарные классы и модульные стили для максимальной гибкости.  
- **React + Next.js:** компоненты разработаны с учетом масштабируемости и переиспользования.  

---

## Технологии

- React  
- Next.js  
- Tailwind CSS  
- SCSS модули  
- react-icons для иконок
